## 📚 Descripción del Proyecto
Este repositorio contiene una implementación práctica de los conceptos fundamentales de la programación en Python, siguiendo el plan de estudios de **Cisco Networking Academy** (Fundamentos de Python). El objetivo es consolidar las habilidades básicas necesarias para desarrollar y comprender programas simples, preparando al estudiante para certificaciones de nivel de entrada como PCEP.

Este proyecto actúa como una colección de laboratorios que demuestran la aplicación práctica de los elementos clave del lenguaje Python.

---

## 🧠 Conceptos Clave Aprendidos (Resumen)
El desarrollo del repositorio cubre el ciclo completo de programación en Python, desde la sintaxis básica hasta las estructuras de control avanzadas:

*   **Introducción a la Programación:** Entendimiento del flujo lógico y los conceptos universales de la informática.
*   **Variables y Datos:** Gestión de tipos de datos (enteros, flotantes, cadenas) y operaciones básicas de Entrada/Salida (`input`/`print`).
*   **Control de Flujo:** Implementación lógica mediante:
    *   **Condicionales:** Toma de decisiones (`if`, `elif`, `else`).
    *   **Bucles:** Repetición controlada de tareas (`for` y `while`).
*   **Estructuras de Datos Avanzadas:** Manipulación organizada de información mediante:
    *   Listas (colecciones ordenadas).
    *   Tuplas, Diccionarios.
*   **Programación Modular:** Creación y uso de funciones personalizadas para reutilizar código y manejar errores (`try`/`except`).

## 📁 Estructura del Repositorio
El proyecto está organizado por módulos que siguen la progresión lógica del aprendizaje:

| Carpeta/Módulo | Contenido Principal | Conceptos Cubiertos |
| :--- | :--- | :--- |
| `modulo-01-introduccion/` | Notas y primer lab. | Sintaxis básica, el programa "Hola Mundo". |
| `modulo-02-variables-y-datos/` | Operadores de lógica. | Tipos de datos, operaciones aritméticas (I/O). |

| `proyecto-final/` | Ejercicios integradores. | Aplicación de todos los conocimientos (funciones, diccionarios). |

## 🛠️ Cómo Ejecutar el Código
Para ejecutar los laboratorios, sigue estos pasos:
1. **Instalar Python:** Asegúrate de tener una versión reciente de Python instalada en tu sistema.
2. **Navegar al Proyecto:** Abre la terminal y dirígete a la carpeta principal: `cd python-netacad-basics`.
3. **Ejecutar Módulos:** Ejecuta cada archivo con el comando: `python módulo_x/lab_y_z.py`
