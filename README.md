# 🐍 Fundamentos de Python 1 - Cisco Networking Academy

Este repositorio contiene mis apuntes, prácticas y resoluciones de los laboratorios del curso **Fundamentos de Python 1 (Python Essentials 1)** impartido por Cisco Networking Academy. 

## 📚 Estructura del Repositorio

El contenido está organizado siguiendo el programa oficial del curso para facilitar la consulta y el estudio progresivo:

- 📁 **[Módulo 1: Introducción a Python y la Programación Informática](./modulo-01)**
  - Conceptos básicos de programación, cómo funciona Python y configuración del entorno.

- 📁 **[Módulo 2: Tipos de datos, Variables, Operaciones Básicas de Entrada y Salida, Operadores Básicos](./modulo-02)**
  - Manejo de variables, tipos de datos fundamentales y la interacción básica con el usuario mediante consola.

- 📁 **[Módulo 3: Valores Booleanos, Ejecución Condicional, Bucles, Listas y su procesamiento, Operaciones Lógicas y de Bit a Bit](./modulo-03)**
  - Control de flujo (`if`, `while`, `for`), estructuras de datos como listas y lógica de programación fundamental.

- 📁 **[Módulo 4: Funciones, Tuplas, Diccionarios, Excepciones y Procesamiento de Datos](./modulo-04)**
  - Creación de funciones propias, manejo de estructuras de datos inmutables y diccionarios, y captura de errores (excepciones).

- 🚀 **[Proyecto Final](./proyecto-final)**
  - Desarrollo del proyecto integrador que pone a prueba todos los conocimientos adquiridos durante los 4 módulos.

## 🎯 Objetivo

Documentar mi progreso y consolidar mis bases en Python como parte de mi formación en Ingeniería en Sistemas Computacionales en el ITSVA y mi crecimiento como desarrollador full-stack. Todo este material sirve además como preparación directa para el **Examen de certificación PCEP – Programador Python certificado de nivel inicial**.
